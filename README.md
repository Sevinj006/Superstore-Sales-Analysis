# Superstore Sales Analysis
Data analysis project based on the Superstore dataset using Python, Excel, and Power BI. Includes sales analysis, profit insights, and interactive dashboard visualization.
#Superstore Sales Analysis Project

# Overview
This project is based on the Superstore dataset originally stored in Excel format.  
It contains detailed retail data including orders, customers, products, shipping details, and financial performance.

The goal of this project is to analyze sales performance and extract meaningful business insights using Python, Excel, and Power BI.

---

# Tools Used
- Microsoft Excel (data source and initial structure)
- Python (pandas) for data analysis
- Power BI for data visualization

---

#Key Analysis Performed (Python)

Using Python (pandas), the following insights were extracted:

-  Average delivery days
-  Number of unprofitable (loss) orders
-  Total profit by category
-  Top 10 customers by sales

---

# Power BI Dashboard

An interactive dashboard was created in Power BI showing:

- Total sales by customer name (Sum of Sales by Customer Name)
- Customer-level performance overview
- Sales distribution insights

---

# Key Insights

- Some categories generate high sales but lower profit margins
- A small group of customers contributes a large portion of total sales
- Delivery time varies across orders and impacts operational efficiency
- Loss-making orders highlight areas for optimization

- Exploratory data analysis (EDA)
- Grouping and aggregation in Python
- Business insight extraction
- Data visualization in Power BI
